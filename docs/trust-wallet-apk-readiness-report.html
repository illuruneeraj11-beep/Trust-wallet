Current verdict

The app is ready for controlled Android QA as a testnet demonstration wallet. EAS preview version 0.3.1/build 13 was built for all four Android ABIs, downloaded, cryptographically verified, installed, cold-launched, and navigated against the connected Supabase and live-market configuration. It is not ready to be represented as the official Trust Wallet or as a real self-custody wallet: it does not hold private keys, sign blockchain transactions, or transfer redeemable cryptocurrency.

Important: keeping the Testnet/unofficial identity is a release requirement. Removing it to make users believe this is the official financial app would create a deceptive and unsafe product.

What still lacks fidelity

  1. Generic email/password first launch and gray splash differ from the recording.
  2. Markets range chips still display 24-hour row changes.
  3. Historical charts and recorded LIVE/1m/1H ranges are incomplete.
  4. Home customization, Perps controls, and several Settings destinations remain simplified.
  5. Receive/Buy loading states and Add Wallet illustration need another visual pass.
  6. The dApp page remains deliberately read-only rather than an unrestricted Web3 browser.
  7. Large-value receipts still show ungrouped digits, one long recipient-address state clips at the left edge, and the lowest review action is close to Android's gesture area.
  8. The visual evidence samples Home, providers, and transaction states; it is not an exhaustive certification of every official-app screen.

Market prices: Exa vs. CoinMarketCap

Do not use Exa for runtime prices. Exa is suitable for researching names, contracts, documentation, and official artwork. Current rates should continue coming from the deployed market endpoint.

  • Verified now: GET https://expo-wallet.vercel.app/api/markets?currency=USD → HTTP 200, JSON, stale=false, 38 quotes.
  • For historical charts and faster streaming, add COINMARKETCAP_API_KEY only as a Vercel Preview/Production secret.
  • The APK should call https://expo-wallet.vercel.app through EXPO_PUBLIC_MARKET_API_BASE_URL; it must never contain the private CMC key.
